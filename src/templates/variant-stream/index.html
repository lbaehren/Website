The Variant Stream template

This is Variant Stream, a free website template by Andreas Viklund. It is written using standards-compliant XHTML 1.0 Strict and CSS 2.1, and it can serve as a starting point for building your own website. The code is simple and well-structured, and the template works in all major browsers - including those on mobile devices such as phones and tablet computers.

For more free templates, web design tutorials and related material, visit andreasviklund.com. If you have any questions, comments or ideas regarding this template, feel free to use the andreasviklund.com discussion forum to get in touch with the template designer and with other template users around the world.
